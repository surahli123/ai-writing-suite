# Reference Prompt Catalog

This package uses public prompt and goal-workflow projects as inspiration while keeping attribution explicit.

## Referenced Projects

- `michaelpersonal/goal-forge`: adapted base concept for turning rough work into autonomous goal contracts.
- `grp06/goalcraft`: goal decomposition inspiration.
- `tolibear/goalbuddy`: lightweight goal refinement inspiration.
- `majiayu000/awesome-goal-prompts`: CC0 prompt-pattern catalog inspiration.
- `lidangzzz/goal-driven`: conceptual inspiration only.

## Reuse Boundary

Use project ideas as reference patterns. Do not copy code, private local runner details, or license-unclear content into public package files.
